## 什么是三斜线指令？有哪些三斜杠指令？

<Answer>


三斜线指令是单行注释，包含用作编译器指令的 XML 标记。每个指令都表示在编译过程中要加载的内容。三斜杠指令仅在其文件的顶部工作，并且将被视为文件中其他任何地方的普通注释。

- `/// <reference path=”…” />` 是最常见的指令，定义文件之间的依赖关系。
- `/// <reference types=”…” />` 类似于path但定义了包的依赖项。
- `/// <reference lib=”…” />` 允许显式包含内置lib文件。

</Answer>
